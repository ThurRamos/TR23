# MICHAEL JACKSON.

## MAIOR CANTOR E COMPOSITOR DE MUSICAS.

### Michael Jackson (1958–2009) foi um cantor, compositor, dançarino e filantropo estadunidense, aclamado mundialmente como o "Rei do Pop". Ele é considerado uma das figuras culturais mais importantes do século XX e o artista de maior sucesso de todos os tempos, tendo revolucionado a indústria musical, a dança e os videoclipes.
